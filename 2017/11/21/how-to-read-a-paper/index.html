如何阅读一篇论文

By JoyKing

学术研究人员通常需要花费大量的时间来阅读论文，学会如何正确有效地阅读论文，不仅能够帮助我们更好的理解论文的核心要点，还能够帮我们节省很多时间和精力。

介绍

在刚开始研究生课程时候，很多课程都会要求我们读很多论文，一开始拿到论文并不能很好的处理他们。在阅读了 How to Read a Paper 这篇文章后，对于如何正确阅读一篇论文有了更好的方法，下面就把这种方法总结一下。

Three-Pass 方法

简单来说，在我们从头至尾通篇阅读论文之前，可以尝试从不同关注点将文章自己处理成三个递进的层次。第一部分关注论文给我们的直观概念和基本思想，第二部分关注论文的内容但并非是所有的细节，第三部分关注论文更深层次的理解。

First Pass

第一部分我们只需要花上 5-10 分钟来很快的把整个文章略读一遍，当然你也可以在这个环节多花几次，来更好的理解论文的基本思想。阅读的步骤如下：

  • 认真地阅读题目(Title)、摘要(Abstract)和介绍(Introduction)
  • 阅读每一章节(Section)和子章节(Sub-Section)的标题，然后忽略其他内容
  • 阅读结论(Conclusion)
  • 浏览一下参考文献(Reference)，脑海中回忆下其中你已经阅读过的论文

在进行完 First Pass 之后，我们应该可以自问自答下面的一些问题：

  • 类别(Category)：这篇论文是什么类型？是对于现有系统或者模型的分析？还是对于一种新型研究模型的描述？
  • 上下文(Context)：这篇论文和哪些文章有关联性？论文中用到了哪些基础理论来分析现有的问题？
  • 正确性(Correctness)：论文中的各种假设是否合理？
  • 贡献性(Contribution)：论文对于现有研究的贡献有哪些？
  • 明确性(Clarity)：论文是否写的比较清晰明确？

利用这一步骤我们通过从论文中得到的各种信息，我们就可以选择是否继续深入的阅读这篇论文。因为这篇论文可能不和你的研究方向相符，或者是这篇论文写得并不够那么精彩，你无法从这篇论文中获取自己想要的信息。

其实在这里，也给我们一个启发，就是如果一篇论文在这些方面没有做好的话，审稿人可能就会拒掉这篇文章，要是更甚的，读完第一遍之后也没有搞清楚论文的侧重点，那么这篇文章可能就不会再有人愿意继续阅读。

Second Pass

在第二部分的过程中，需要更加认真的阅读论文，但是关于数学证明等繁琐的细节可以忽略掉。我们可以在每一个章节的空白地方增加上自己对于该部分的一个想法。如下：

  • 仔细查看图片、图表和论文中其他可视化描述的部分。
  • 对于那些参考文献中没有阅读过的论文做个标记，这样能够更好的理解当前论文的研究背景

第二部分的过程可能会花费比较多的时间，再进行第二部分的时候，我们应该更关注论文的内容，而且也可以进一步的对于整篇论文的重点部分进行自我总结。

有时候进行完第二部分的阅读，我们可能也无法很好的理解论文中的各种思想，很有可能这篇论文的内容对于你来说是一个比较新的领域，比如我在阅读很多关于计算机网络路由协议的论文时，经常是找不到头绪，但是在读过几篇之后，就会对这个领域有一个浅显的认识，进而理解起来就会比较快了。

另外一种情况就是论文的作者采用了大量的证明或者你并不太了解的实验，关于这个问题，文章也给出了一些建议：

  • 直接将论文放在一边，因为很有可能你以后的方向并不会很需要这篇论文的知识
  • 了解一些该论文的研究背景，再回过头来阅读论文
  • 或者是坚持下去，完成下一步骤的阅读

Third Pass

通常来说第三部分需要花费的时间会更长，而且第三部分我们可以直观的理解为对于一篇论文的升华。第一部分的时候，我们对于论文的基本情况，论文类型、基本内容、结论等各个方面有了解，通过第二部分，我们能够进一步的看到论文的侧重部分，也就是对于第一部分提取的框架，进行实体内容的填充。

用原文的话来说，第三部分我们需要进行 virtually re-implement，也就是利用前面第一二部分了解到的知识，自己在脑海中对于论文进行重新的组织，根据论文作者提出的各种假设，来将这些已经做好的工作重新组织一遍。

可能这是一个比较抽象的过程，但是通过这样的一个过程，我们可以关注一下自己重新组织的内容和论文原本内容的差异，以及在做一些假设检验或者理论证明的时候，我们是如何思考，而论文是如何解决这样的问题。这样我们就能更深入的理解论文的整个结构，无论是从一开始问题的提出，到设想，再到理论及实验，以及最后的结论。同时我们也能更好的体会作者为什么会对于未来的工作作出这样的设想。

总结

这篇大概就是对于怎么去阅读一篇文章的方法进行了个整理，未来要阅读的论文还有很多很多，在把有限的生命投入到无限的学习中之前，选择对于我们有益的论文或者书籍的重要性远大于随随便拿到一篇论文或者一本书籍就开始无脑读下去。

如果有兴趣进行学术研究，那么你可能会需要这篇关于如何写论文的文章。
